# SHIFT: Key User Stories

*   **As a Small Business Owner,** I want to connect my Shopify store to QuickBooks Online so that new orders are automatically created as invoices without manual data entry.
*   **As a Developer,** I want to create a custom API endpoint on a self-hosted runner that transforms incoming webhook data before forwarding it to my internal database.
*   **As an IT Admin,** I want to add a second runner to my production cluster to ensure that if one server goes down for maintenance, my integrations continue to run without interruption.
*   **As a Security Officer,** I want to enforce that all custom APIs exposed by our runners authenticate clients using our company's Azure AD (OIDC).
